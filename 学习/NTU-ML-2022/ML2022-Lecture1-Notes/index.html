What is Machine Learning?

Machine Learning ≈ Looking for Function

Different types

  • Regression(回归): 输出要预测的数值
  • Classification(分类): 输出给定的options中的某一个
  • Structured Learing: 创造有结构的物件(作图,写文章)

从最基本的Linear Model开始

例: 预测youtube频道第二天播放量

构建模型

基本术语:

  • model: 带有未知参数的函数, 这里就是y=b+wx1y = b + wx_1y=b+wx1​
  • feature: x1x_1x1​, 已知的数据
  • weight, bias: 分别是www和bbb

定义损失函数

一般可以表示为:

L=1N∑nenL = \frac{1}{N}\sum_{n}e_n L=N1​n∑​en​

其中e代表每组数据预测值跟真实值的误差, e可以有不同的计算方法, 如MAE/MSE等。

Error Surface

最优化(optimization)

梯度下降(gradient descent)

  1. 随机挑选初始值w0,b0w^0,b^0w0,b0
  2. 更新参数:

w1←w0−η∂L∂w∣w=w0,b=b0b1←b0−η∂L∂b∣w=w0,b=b0\begin{align*} w^1 &\leftarrow w^0 - \eta \left. \frac{\partial L}{\partial w} \right|_{w=w^0, b=b^0} \\ b^1 &\leftarrow b^0 - \eta \left. \frac{\partial L}{\partial b} \right|_{w=w^0, b=b^0} \end{align*} w1b1​←w0−η∂w∂L​​w=w0,b=b0​←b0−η∂b∂L​​w=w0,b=b0​​

  1. 迭代

其中η\etaη就是学习率(learning rate), 一般要手动设定。
我们把ML中需要手动设定的参数称为hyperparameter

关于激活函数


大部分复杂的函数图像理论上都可以使用大量的蓝色折线图叠加来表达
但是我们又该怎么表示这个「蓝色的折线图」呢?
🤓☝️我们有sigmoid函数

y=c11+e−(b+wx1)=c sigmoid(b+wx1)\begin{align*} y &= c \frac{1}{1 + e^{-(b + wx_1)}} \\ &= c\,sigmoid(b + wx_1) \end{align*} y​=c1+e−(b+wx1​)1​=csigmoid(b+wx1​)​

而实际上这条「蓝色的折线图」就通常被称为「Hard Sigmoid」
